Einfache Beispiele

Diese Beispiele sind perfekt für den Einstieg. Sie zeigen grundlegende Konzepte, die leicht zu verstehen sind.

Beispiel: Visitenkarte

Sehr einfach

Eine digitale Visitenkarte mit Name, Kontaktdaten und Design. Perfekt um HTML-Struktur und CSS-Styling zu lernen.

Was du lernst:

  • HTML-Grundstruktur (div, h1, p)
  • CSS-Layout (Flexbox, Zentrierung)
  • Farbverläufe (linear-gradient)
  • Hover-Effekte
  • Box-Shadow für Tiefe

Kern-Code:

<div class="card">
    <h1>Max Mustermann</h1>
    <p class="job-title">Web-Entwickler</p>
    <p>max@beispiel.at</p>
</div>

Aufgaben zum Experimentieren:

  • Ändere die Farben
  • Füge ein Foto hinzu
  • Ändere Schriftarten (Binde eine Google Font ein)
  • Baue Bewegung mit einer CSS Animation ein
  • Füge Social-Media-Links hinzu

Beispiel: Button-Sammlung

Sehr einfach

Verschiedene Button-Designs mit unterschiedlichen Hover-Effekten. Zeigt die Vielfalt von CSS.

Was du lernst:

  • CSS-Transitions (sanfte Übergänge)
  • Hover-States (:hover)
  • Transform (verschieben, skalieren, drehen)
  • Box-Shadow und Glow-Effekte
  • Border und border-radius

Kern-Code:

.btn-classic {
    background: #3498db;
    color: white;
    transition: all 0.3s;
}

.btn-classic:hover {
    background: #2980b9;
    transform: translateY(-2px);
}

Aufgaben zum Experimentieren:

  • Erstelle deinen eigenen Button-Style
  • Ändere die Animations-Geschwindigkeit
  • Probiere verschiedene Transform-Effekte aus
  • Kombiniere mehrere Effekte

Mittlere Beispiele

Diese Beispiele kombinieren HTML, CSS und JavaScript. Sie sind interaktiv und reagieren auf Benutzereingaben.

Beispiel: Quiz

Mittel

Ein interaktives Quiz mit Multiple-Choice-Fragen. Zeigt JavaScript-Grundlagen und Event-Handling.

Was du lernst:

  • Variablen (let, const)
  • Funktionen definieren und aufrufen
  • If/Else-Bedingungen
  • DOM-Manipulation (classList, textContent)
  • Event-Handling (onclick)

Kern-Code:

function checkAnswer(element, isCorrect) {
    if (isCorrect) {
        element.classList.add('correct');
        correctAnswers++;
    } else {
        element.classList.add('wrong');
    }
    showResult();
}

Aufgaben zum Experimentieren:

  • Füge neue Fragen hinzu
  • Ändere das Thema (Sport, Gaming, Filme)
  • Füge einen Timer hinzu
  • Erstelle Schwierigkeitsstufen

Beispiel: Zitate-Generator

Mittel

Zeigt zufällige Motivations-Zitate an. Demonstriert Datenstrukturen und Zufallszahlen.

Was du lernst:

  • Arrays (Listen von Daten)
  • Objekte (strukturierte Daten)
  • Math.random() für Zufallszahlen
  • Animations-Übergänge mit Opacity
  • setTimeout() für Verzögerungen

Kern-Code:

const quotes = [
    { text: "Glaube an dich...", author: "Unbekannt" },
    { text: "Der Weg zum Erfolg...", author: "Goethe" }

const randomIndex = Math.floor(Math.random() * quotes.length);
const quote = quotes[randomIndex];

Debugging-Tipps

Fehler sind normal und gehören zum Programmieren dazu. Hier sind einige Tipps zum Finden und Beheben von Fehlern:

1. Browser-Console öffnen

Drücke F12 (oder Rechtsklick → Untersuchen). Die Console zeigt Fehler und Warnungen an.

2. console.log() verwenden

Füge console.log(variable) ein, um Werte zu überprüfen und den Programmfluss zu verfolgen.

3. Häufige Fehler

  • Vergessene Semikolons
  • Nicht geschlossene Klammern
  • Groß-/Kleinschreibung
  • Nicht geschlossene Anführungszeichen

4. Google ist dein Freund

Kopiere die Fehlermeldung und google sie. Websites wie Stack Overflow haben meist die Antwort.
