Grundlagen der Webentwicklung

Die wichtigsten Konzepte und Begriffe verstehen

Was ist Coding?

Coding (oder Programmieren) bedeutet, einem Computer präzise Anweisungen zu geben, damit er bestimmte Aufgaben ausführt. Es ist wie das Schreiben eines sehr detaillierten Rezepts oder eines Bauplans.

Beispiel

Stell dir vor, du musst einem Roboter erklären, wie man ein Marmeladebrot macht. Du müsstest jeden einzelnen Schritt genau aufschreiben: "Nimm das Brot", "Öffne das Glas", "Nimm das Messer" usw. Beim Programmieren ist es ähnlich - wir müssen dem Computer jeden Schritt erklären.

Die drei Säulen einer Website

Jede moderne Website besteht aus drei Hauptkomponenten, die zusammenarbeiten:

HTML

Die Struktur

HTML (HyperText Markup Language) ist das Grundgerüst. Es definiert, welche Elemente auf der Seite vorhanden sind.

Wichtige HTML-Elemente:

  • <h1> bis <h6> - Überschriften
  • <p> - Absätze (Paragraphs)
  • <div> - Container für andere Elemente
  • <button> - Schaltflächen
  • <img> - Bilder
  • <a> - Links

Beispiel:

<h1>Willkommen</h1>
<p>Das ist ein Absatz.</p>
<button>Klick mich</button>

CSS

Das Design

CSS (Cascading Style Sheets) gestaltet die Website. Es bestimmt Farben, Schriften, Abstände und das gesamte Aussehen.

Wichtige CSS-Eigenschaften:

  • color - Textfarbe
  • background - Hintergrundfarbe
  • font-size - Schriftgröße
  • padding - Innenabstand
  • margin - Außenabstand
  • border - Rahmen

Beispiel:

h1 {
    color: blue;
    font-size: 32px;
}

JavaScript

Das Verhalten

JavaScript macht Websites interaktiv. Es kann auf Benutzeraktionen reagieren, Berechnungen durchführen und Inhalte dynamisch ändern.

Wichtige JavaScript-Konzepte:

  • Variablen - Daten speichern
  • Funktionen - Code wiederverwenden
  • Events - Auf Aktionen reagieren
  • Bedingungen - Entscheidungen treffen
  • Schleifen - Wiederholungen

Beispiel:

function sayHello() {
    alert('Hallo Welt!');
}

Außerdem:

Außerdem haben größere Webseiten und Seiten mit denen Interaktion möglich ist (z.B. Webshops, Messenger, Webseiten mit CMS-Systemen, ...) auch noch serverseitigen Code, Datenbanken, ... Damit befassen wir uns heute allerdings noch nicht.

Wichtige Begriffe

Tag

Ein HTML-Befehl in spitzen Klammern

<p>, <div>, <button>

Element

Ein vollständiger HTML-Baustein mit öffnendem und schließendem Tag

<p>Text hier</p>

Attribut

Zusätzliche Information für ein Element

<div class="container">

Selektor

CSS-Adresse, um Elemente anzusprechen

.container { ... }

Eigenschaft

Eine CSS-Styling-Regel

color: red;

Funktion

Wiederverwendbarer JavaScript-Code

function name() { ... }

Variable

Ein Speicherplatz für Daten

let name = "Max";

Event

Eine Benutzeraktion wie Klicken

onclick="..."

DOM

Das Document Object Model verbindet HTML mit JavaScript, indem es die Struktur eines Dokuments als Baumstruktur im Speicher abbildet.

Bugs

Fehler im Code. Ganz zu Beginn waren damit tatsächlich Käfer gemeint!

Debugging

Fehler suchen un beheben. Eine der Hauptaufgaben eines Programmierers.

"Vibe Coding"

Programmieren mit LLMs (ChatGPT und Co). Nützlich für alle, die sich schon auskennen. Kann aber zu viel Debuggen führen.

Entwicklungsumgebung

Um selbst Websites zu erstellen, brauchst du nur zwei Dinge:

1. Browser

Zum Anzeigen und Testen deiner Website

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari
Tipp: Die meisten Browser haben eingebaute Entwicklertools (F12).

2. Texteditor

Zum Schreiben deines Codes

  • Visual Studio Code
  • Webstorm
  • Notepad++
  • Atom
Tipp: Macht euch mit eurer Entwicklungsumgebung vertraut! VS Studio und Webstorm bieten viele nützliche Features.

Alternative: Online-Editoren

Wenn du nichts installieren möchtest, kannst du auch Online-Tools verwenden wie CodePen.io, JSFiddle.net oder Replit.com. Diese funktionieren direkt im Browser.
