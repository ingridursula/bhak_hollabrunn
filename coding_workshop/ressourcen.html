Ressourcen

Tools, Dokumentationen und Lernplattformen für deine Coding-Reise

Online-Editoren

Mit diesen Tools kannst du direkt im Browser Code schreiben und testen, ohne etwas zu installieren.

CodePen

codepen.io

Der beliebteste Online-Editor für HTML, CSS und JavaScript. Perfekt zum Experimentieren und Teilen von Code.

Vorteile:

  • Live-Vorschau während du tippst
  • Große Community mit vielen Beispielen
  • Kostenlos und ohne Anmeldung nutzbar
  • Projekte können gespeichert und geteilt werden
Anfänger-freundlich Kostenlos

JSFiddle

jsfiddle.net

Ähnlich wie CodePen, mit einfacher Oberfläche und schnellem Setup.

Vorteile:

  • Sehr einfache Bedienung
  • Schnell zum Loslegen
  • Kann externe Bibliotheken einbinden
  • Kollaboratives Arbeiten möglich
Anfänger-freundlich Kostenlos

JS Bin

jsbin.com

Minimalistischer Online-Editor mit Fokus auf JavaScript-Debugging.

Vorteile:

  • Sehr übersichtliche Oberfläche
  • Integrierte Console für JavaScript
  • Versionierung der Änderungen
  • Extrem schnell
Anfänger-freundlich Kostenlos

Dokumentationen und Referenzen

Diese Websites sind wie Nachschlagewerke für HTML, CSS und JavaScript. Hier findest du alle Informationen zu Tags, Eigenschaften und Funktionen.

W3Schools

w3schools.com

Die wahrscheinlich umfassendste Lern-Ressource für Webentwicklung. Auch auf Deutsch verfügbar.

Inhalt:

  • Tutorials für HTML, CSS, JavaScript
  • Interaktive Beispiele zum Ausprobieren
  • Übungen und Quizzes
  • Vollständige Referenz aller HTML-Tags und CSS-Eigenschaften
Anfänger-freundlich Deutsch verfügbar Kostenlos

MDN Web Docs

developer.mozilla.org

Die offizielle Dokumentation von Mozilla. Sehr detailliert und technisch korrekt.

Inhalt:

  • Detaillierte Erklärungen zu allen Web-Technologien
  • Best Practices und Tipps
  • Browser-Kompatibilitätstabellen
  • Fortgeschrittene Themen
Für Fortgeschrittene Kostenlos

css-tricks.com

Inhalt:

  • CSS-Tutorials und Guides

  • Video-Tutorials

Lernplattformen

Diese Plattformen bieten strukturierte Kurse und interaktive Übungen zum Lernen von Programmierung.

Selfhtml JavaScript Tutorials

wiki.selfhtml.org/wiki/JavaScript

Kostenlose Schritt-für-Schritt-Anleitungen für konkrete Problemstellungen

Kurse:

  • Einstieg in JavaScript
  • JavaScript und DOM
  • Spieleprogrammierung Tutorials
  • Fortgeschrittene JavaScript Konzepte

Dauer: ca. 10 Stunden für die Anfänger Beispiele

Level: Blutige Anfänger bis Fortgeschritten

Kostenlos Ohne Anmeldung

freeCodeCamp

freecodecamp.org

Komplett kostenlose Lernplattform mit umfassenden Kursen zur Webentwicklung.

Kurse:

  • Responsive Web Design (HTML & CSS)
  • JavaScript Algorithms and Data Structures
  • Frontend Development Libraries
  • Projekte zum Portfolio aufbauen

Dauer: Mehrere hundert Stunden Inhalt

Level: Anfänger bis Fortgeschritten

Kostenlos Zertifikat Interaktiv

Codecademy

codecademy.com

Interaktive Coding-Kurse im Browser. Die Grundlagen sind kostenlos verfügbar.

Kurse:

  • Web Design & Development Pfade

Dauer: Kurse von 10-50 Stunden

Level: Anfänger bis Mittelstufe

Community und Hilfe

Wenn du nicht weiterkommst, helfen dir diese Communities:

Stack Overflow

stackoverflow.com

Die größte Frage-und-Antwort-Plattform für Programmierer. Fast jede Fehlermeldung wurde hier schon einmal diskutiert.

Tipp: Kopiere deine Fehlermeldung und suche danach auf Stack Overflow.

Reddit - r/learnprogramming

reddit.com/r/learnprogramming

Freundliche Community für Anfänger. Du kannst Fragen stellen und bekommst meist schnell Hilfe.

Reddit - r/webdev

reddit.com/r/webdev

Speziell für Webentwicklung. Hier werden aktuelle Trends diskutiert und Hilfe angeboten.

Discord - The Programmer's Hangout

Discord-Server mit Tausenden von Entwicklern. Live-Chat für schnelle Hilfe.
