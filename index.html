CSS Flexbox-Modell

Das Flexbox-Layout (display: flex;) ist eine leistungsstarke Möglichkeit, Elemente flexibel und dynamisch anzuordnen. Es eignet sich besonders gut für responsives Design und die Verteilung von Inhalten innerhalb eines Containers.

HTML5 Animationen

Mit HTML, CSS und JS hat man alle Werkzeuge beisammen, um damit "einfache" Animationen im Webbrowser zu bauen.

Aufwändigere Animationen benötigen allerdings viele hundert Zeilen Code. Auch wenn man das in einem Editor wie WebStorm selbst schreiben könnte, nimmt man lieber Programme wie zum Beispiel Adobe Animate dafür.

Damit lassen sich Animationen, Werbebanner und auch Spiele erstellen, und als Webinhalte (HTML, CSS, JS) exportieren, die man dann im Browser öffnen kann.

  • Bubblecano
  • Github - Sourcecode zu Bubblecano
  • YouTube - Adobe Animate Crash Course
  • HTML5 Spiele
